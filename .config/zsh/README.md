## Installation

```bash
./setup.zsh
build-fzf-tab-module # NOTE: see https://github.com/Aloxaf/fzf-tab?tab=readme-ov-file#binary-module
fast-theme base16    # or whatever fast-syntax-highlighting theme
```

## Keybindings

- Move forward (word):
  `Ctrl-F`
- Move previous (word):
  `Ctrl-B`
- Move forward (char):
  `Alt-F`
- Move previous (char):
  `Alt-B`
- Toggle sudo:
  `Ctrl-X`
- Toggle process bg/fg:
  `Ctrl-Z`
- Accept autosuggestion:
  `Ctrl-E`
- Repeat last command:
  `Ctrl-K`
- FZF files/dir search:
  `Ctrl-T`
- FZF history search:
  `Ctrl-R`
